# Noir Studio

Modern salon booking website with AI chatbot assistant.

## Features

- Online booking system
- AI chatbot assistant
- Admin dashboard
- Service management
- Real-time available slots

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- Supabase
- Gemini AI

## Installation

Clone the repository:

```bash
git clone https://github.com/Azmirabbani/noir-studio.git
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```
